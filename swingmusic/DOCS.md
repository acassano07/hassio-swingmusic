# Swing Music Home Assistant Addon

[Swing Music](https://github.com/swingmx/swingmusic) is a beautiful, self-hosted music player for your local music collection.

## Features
-   Modern, clean UI.
-   Self-hosted and private.
-   Fast scanning and playback.
-   Ingress support for Home Assistant.

## Installation
1.  Add this repository URL to your Home Assistant Add-on Store.
2.  Install the "Swing Music" add-on.
3.  Ensure your music is in the `/media` folder (default).

## Configuration
-   **Music Path**: Where the addon looks for music. Defaults to `/media`.

## Default Credentials
-   **User**: `admin`
-   **Password**: `admin`
-   *Please change these immediately after logging in.*

## How to use
-   Open the addon from the sidebar.
-   Log in with the default credentials.
-   Add your music folder (usually `/media`) in the app's settings.
